6. 链接   四个链接状态是：    a:hover 必须跟在 a:link 和 a:visited后面，a:active 必须跟在 a:hover后面
   - a:link - 正常，未访问过的链接
   - a:visited - 用户已访问过的链接
   - a:hover - 当用户鼠标放在链接上时
   - a:active - 链接被点击的那一刻
