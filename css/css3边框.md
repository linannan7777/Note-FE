## 新边框属性

| 属性          | 说明                                           | CSS  |
| :------------ | :--------------------------------------------- | :--- |
| border-image  | 设置所有边框图像的速记属性。                   | 3    |
| border-radius | 一个用于设置所有四个边框- *-半径属性的速记属性 | 3    |
| box-shadow    | 附加一个或多个下拉框的阴影                     | 3    |
